# AIS API key creation

1) Head to https://www.barentswatch.no/ and create a user.
2) Go to your profile page https://www.barentswatch.no/minside/ and scroll down until you see "Developer access".
3) Click "Show more" under AIS - API.
4) Click "New Client" and fill in the fields.
5) AIS_CLIENT_ID is the Client ID under "My clients", and the AIS_CLIENT_SECRET is the password used to create the corresponding client.
